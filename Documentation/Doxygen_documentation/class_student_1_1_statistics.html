Detailed Description

Defines the interface for statistics class.

Invariant
getPoints() >= 0
getCreditBalance() >= MAX_LOAN_ALLOWANCE MAX_LOAN_ALLOWANCE is the limit that the player is allowed to have debt. The actual value is to be decided by the implementer of inherited class.
